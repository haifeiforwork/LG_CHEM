Ⅲ. 제도소개
1. 지원대상
전 사원(단, 임원, 촉탁, 계약직 제외) 으로 해당월(급여기산일기준) 15일이상 근무자
1) 적용대상
- 신규가입자 : 매월 1일까지 입사자는 당월부터, 매월2일 입사자는 익월부터 신청가능
- 휴직자 : 휴직기간중에는 지원하지 않음. (휴직월 : 지원, 복직월 : 근무일수에 따라 지원)
- 산재자 : 근무일수와 상관없이 계속지원
- 산전산후 휴가자 : 기준 미결
2) 지원기간
- 2016.6월까지 가입기간중 월 5만원 지원
3) 적용보험 구분
- 남자 : 20세미만, 46세 이상자 -> 마이라이프
20세이상~45세 이하자 -> 개인연금
- 여자 : 20세미만, 46세 이상자 -> 마이라이프
20세이상~45세 이하자 -> 선택가능
 
2. 기타내용
1) 월보험료 : 5만원(회사5만원 지원) 급여공제 납부
개인연금은 3%할인금액(1,550원)포함 실제 51,550원/월 불입됨
2) 가입기간 : 개인연금은 10년
마이라이프->3년 (만기후 개인연금 가입 또는 기간연장 가능)
3) 만기시 : 만기금을 수령코자하는 통장 COPY본 및 신분증 COPY본을 개인연금 담당자에게 제출
4) 퇴직시 : 희망여부에 따라 계속 가입 가능. (문의 : LG화재 ☎02-6333-6010)
단, 본인이 5만원 전액 부담
 
3. 담보내용(개인연금)
1) 보험가입금액 - 1,000만원
2) 상해사망 - 교통사고 : 2,000만원 + 그때까지의 적립금액 지급
- 기      타 : 1,000만원 + 그때까지의 적립금액 지급
3) 후유장애 - 교통사고 : 60만원 ~ 2,000만원까지의 장애정도에 따라 차등지급
- 기      타 : 30만원 ~ 1,000만원까지의 장애정도에 따라 차등지급
