개요
  개요
    선택한 Job의 Profile 내용을 수정하도록 구성된 화면임.

 화면사용법 및 유의사항
    <그림 1. Job Profile 수정화면, 저장 및 취소 Button있는 화면>
   
 
    1. Function 수정 : 선택된 Job의 Function을 수정하는 것으로, Function List를 Click하면 조회권한이 있는
    Function이 나타나며, 변경하고자 하는 Function을 선택한다.
    2. Objective 수정 : 선택된 Job의 Objective 를 수정하는 것으로, 위에서 Function을 선택하면 해당
    Function에 속하는 Objectives만 나타나며, Objectives List를 Click하여 변경하고자 하는 Objective 를 선택한다.
    3. 대분류 수정 : 선택된 Job의 대분류를 수정하는 것으로 위의 선택된 Objective 에 해당되는 대분류가
    나타나며, 대분류 List를 Click하여 변경하고자 하는 대분류를 선택한다.
    4. Job Holder 지정 : 선택된 Job의 수행자를 수정하기 위한 기능 Button임.
    1) Job Holder 지정 Icon을 Click하면 Job Holder를 새로 지정하는 화면이 Pop-Up 창으로 나타난다.
 
    <그림 2. Job Holder 지정화면>
   
 
    2) 기본적으로 조회권한을 가지는 조직의 구성원들이 성명, 직위, 사번 순으로 Display된다.
    3) 현재 해당 Job을 수행하고 있는 직원들이 Check Box에 V 표시되어 있다.
    4) Check Box 를 Click하여 직무 수행자를 변경한다.
    5) 화면 하단의 지정 Icon을 Click하면 지정이 되면서 해당 Job의 Profile 수정화면으로 되돌아 간다.
    5. Job 시작일 변경 : Job Holder 의 Job 시작일을 Calendar 검색으로 변경할 수 있으며, 새로 지정된
    경우에는 Default로 현재일자를 Display한다.
    6. Job Grade 수정 : 선택된 Job의 Grade를 수정한다.
    1) 현재 화면에 Display되어 있는 Grade를 Click하면 Job Profile 화면에 변경된 사항이 있으면
    "Job Profile을 저장하시겠습니까?"라는 Message 가 뜨고 "예"를 선택하면 저장되며, Job Leveling Sheet 화면으로 이동한다.
 
    <그림 3. Job Leveling Sheet 조회 화면, 수정 Button이 있는 화면>
   
 
    2) 화면 하단의 수정 Button을 Click하면 Job Leveling 수정화면으로 이동한다.
 
    <그림 4. Job Leveling Sheet 수정화면, 저장/취소 Button이 있는 화면>
   
 
    3) 화면에 Display되는 9개의 직무평가요소별 Grade를 선택하면 Job Leveling 결과를 자동으로
    계산하여 나타낸다.
    4) 화면 하단의 Job Leveling 결과 Box의 Job Leveling 결과 List를 Click하여 수동으로도 변경이
    가능하다.
    5) 화면 하단의 취소 Button을 Click하면 조회화면으로 바뀐다.
    6) Job Leveling Sheet 조회화면에서 화면 상단의 Job Profile Button을 선택하여 해당 Job의 Profile
    조회화면으로 이동한다.
    7. 직무목적 수정 : 직무목적 내용을 Text 로 입력할 수 있으며, 수정내용을 직접 입력한다.
    8. 주요 책임 및 활동 수정 : 주요 책임 및 활동 내용을 Text 로 입력할 수 있으며, 수정내용을 직접 입력한다.
    9. 직무요건 수정 : 직무요건 내용(지식, 스킬, 경험, 태도)을 Text 로 입력할 수 있으며, 수정 내용을 직접
    입력한다.
    10. 수정작업을 완료하면 반드시 화면 하단의 저장 Button을 Click해야 하며, 취소 Button을 Click 하면
    해당 Job의 Profile 조회화면으로 바뀐다.
    11. Competency Requirements 내용을 수정하기 위하여는 화면 상단의 Competency Requirements
    Button을 Click하여 이동한다.
