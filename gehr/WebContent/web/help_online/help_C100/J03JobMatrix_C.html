개요
  개요
    Job Description 내용(Job Profile, Competency Requirements, Job Unit별 KSEA, Job Process 및 Job Leveling Sheet)을 수정/생성/삭제하기 위한 초기화면으로 구성됨.
 
  용어 설명
    1) Job 생성 : 팀 내에서 기존에 수행하던 Job 이외에 새로운 Job에 대한 Job Description 내용 등록을 위한
    기능임.
    2) 대분류 생성 : 업무의 본질적인 특성이나 수행결과의 유사성을 중심으로 팀 내 수행되고 있는 Job들의
    새로운 대분류를 생성하기 위한 기능임.
    3) Job 이동 : Matrix에 나타난 Job들의 대분류, Objectives의 변경(이동)을 위한 기능임.

 화면사용법 및 유의사항
    <그림 1. Job Description 수정.생성.삭제 초기화면(Job Matrix) 조회화면>
   
 
  Job Description 수정.삭제
    1) Matrix 하단에 "※ Job을 조회/수정/삭제하시려면 해당 Job을 Click해 주세요." 라는 Message가 있음.
    2) 수정하거나 삭제하려는 Job을 Matrix 상에서 선택하면 해당 Job Profile의 조회화면으로 이동한다.
 
  대분류 수정.삭제
    1) Matrix 하단에 "※ 대분류를 조회/수정/삭제하시려면 해당 대분류를 Click해 주세요."라는 Message가
    있음.
    2) 수정하거나 삭제하려는 대분류를 Matrix 상에서 선택하면 대분류를 조회하는 화면으로 이동한다.
 
  Job 생성
    1) Matrix 하단에 Job 생성 Button을 선택하면, Job을 신규로 생성하고 팀원에게 지정하는 화면으로 이동
    한다.
 
  대분류 생성
    1) Matrix 하단에 대분류 생성 Button을 선택하면, 대분류를 생성하는 화면으로 이동한다.
 
  Job 이동
    1) Matrix 하단에 Job 이동 Button을 선택하면, Job의 대분류를 변경(이동)하는 화면으로 이동한다.
