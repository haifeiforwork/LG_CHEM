개요
  개요
    해당 대분류와 관련한 모든 데이터 (Job 지정 포함) 를 삭제할 수 있도록 구성된 화면임.

 화면사용법 및 유의사항
    <그림 1. 대분류명 삭제화면>
   
 
    1. 대분류 삭제화면은 대분류 조회화면 하단의 대분류 삭제 Button을 Click하면 Pop-Up 창으로 나타난다.
    2. 대분류 시작일에 최초 시작일이 Display 된다. 적용일자에는 Default로 현재일자가 Display 이 되며,
    Calendar 검색으로 원하는 날짜를 지정할 수 있다.
    3. 화면에 Display되어 있는 대분류 명칭을 확인하고, 삭제 Button을 Click하면 화면에 "대분류와 관련된
    모든 데이터가 삭제됩니다. 삭제하시겠습니까?" 라는 경고 Message가 나타난다.
    4. 삭제 Button 을 Click하면 모든 데이터는 삭제한다.
