개요
  개요
    Competency List는 직무 수행에 있어 업무목표 달성을 위해 요구되는 역량인 지식, 기술, 태도/행동양식 등 직무요건과 요구역량 들을 Dictionary 형태로 조회할 수 있도록 구성된 화면임.

 화면사용법 및 유의사항
    <그림 1. Competency List 조회화면>
   
 
  Competency 조회 방법
    1) Elian 초기화면에서 Job Description 관리 → Competency List 를 선택하면 Data Base화 된 Competency
    를 조회할 수 있다.
    2) Competency의 조회는 한글 자음, 영문 과 Key Word 입력의 3가지 방법으로 할 수 있다. 한글 자음과
    영문 알파벳을 중 하나를 선택하면, 해당 자음과 영문을 첫 글자로 하는 Competency List가 화면에
    나타난다.
    3) Key Word를 입력하면 해당 Key Word를 Competency 명에 포함하고 있는 Competency List 가 화면에
    나타난다.
 
    <그림 2. Competency 상세조회 화면>
   
 
  Competency 상세 내용 조회
    1) 위의 3가지 방법을 통해 화면에 나타난 Competency List 상의 Competency 명을 선택하면 해당
    Competency의 상세 내용을 조회할 수 있는 화면으로 이동한다.
    2) 동 화면에서 다시 위의 3가지 방법을 통해 다른 Competency 를 조회할 수 있다.
    3) 화면 하단의 ‘목록’ Icon을 선택하면 List 화면으로 되돌아 간다.
