개요
  개요
    선택한 Job의 Competency 내용을 수정하도록 구성된 화면임.

 화면사용법 및 유의사항
    <그림 1. Competency Requirements 조회화면, 수정 Button있는 화면>
   
 
    1. Competency Requirements 조회화면 하단의 수정 Button을 Click하면, 수정화면으로 이동한다.
 
    <그림 2. Competency Requirements 수정화면, 저장/취소 Button있는 화면>
   
 
    2. 요구역량 수정(변경) : 수정하고자 하는 요구역량 명칭을 Click하면 해당 역량의 상세조회 화면이
    Pop-Up 창으로 나타나며, 내용 확인 후 수정사항이 있으면 Pop-Up 창을 닫고 화면 우측의 지정 Button을 Click 한다.
    1) 수정(변경)할 요구역량 화면 우측의 지정 Button을 Click하면, Competency List가 Pop-Up 창으로
    나타난다.
 
    <그림 3. Competency List 화면, 닫기 Button있는 화면>
   
 
    2) Competency List 상에서 Competency 조회 방법에 따라 Competency들을 조회하고, 좌측의 Radio
    Button을 선택하면, 자동으로 Competency Requirements 화면에 변경되어 Display된다. 이때 요구역량 명칭을 Click하면 해당 역량의 상세조회 화면으로 이동한다.
    3) 2)의 방법으로 선택한 역량이 기존에 이미 지정되어 있는 경우에는 "기존에 존재하는 Competency
    입니다." 라는 Message가 나타난다.
    4) 화면 하단의 닫기 Button을 Click하면 Competency Requirements 화면으로 이동한다.
    5) 새로운 요구역량이 지정되면, 해당 역량의 요구수준을 List Box를 이용하여 선택하여야 하며, 이때 Key
    Words와 행동지표는 자동으로 변경된다.
    3. Competency 행 추가 : 기존에 지정되어 있는 것 이외에 또 다른 Competency를 추가한다.
    1) 해당 Competency 영역 Box 하단에 Competency 행 추가를 Click하여 행을 추가한다.
    2) 새로 생긴 행 우측의 지정 Button을 Click하여 2.항과 같은 방법으로 Competency 내용을 추가한다.
    4. Competency 삭제 : 삭제하고자 하는 요구역량의 우측에 있는 삭제 Button을 Click 한다.
    5. 화면 하단의 저장 Button을 Click하여 수정사항을 저장한다.
    6. 취소 Button을 Click하면 Competency Requirements 조회화면으로 이동한다.
