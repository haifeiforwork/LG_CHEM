개요
  개요
    생성된 Job 수행을 위해 요구되는 Common Competency 와 Special Competency를 생성하도록 구성된 초기화면임.

 화면사용법 및 유의사항
    <그림 1. Competency Requirements 생성화면>
   
 
    1. 초기화면에는 생성된 Job의 명칭과 Common Competency와 Special Competency 각각 4개씩의 입력
    칸이 Display 되어 있다.
    2. 생성할 Competency 칸 우측의 지정 button을 click 하면 Competency List 화면이 나타난다.
 
    <그림 2. Competency List 조회화면>
   
 
    3. Competency List 화면에서 Competency 명을 Click하면 해당 Competency의 상세내용을 조회할 수
    있으며, 화면 하단의 목록 Button을 Click하면 Competency List 화면으로 되돌아 간다.
    4. 새로 생성된 Job 수행 시 요구되는 역량을 선택하면, 자동으로 Competency Requirements 생성화면으로
    되돌아 가며, 선택한 Competency 명이 display 된다.
    5. 해당 Competency의 요구수준을 선택하면 Key Words 및 행동지표가 자동으로 Display 된다.
    6. Common과 Special Competency 각각 4개씩 의 역량을 선택한 후 역량을 추가할 경우에는
    ‘Common Competency 행 추가’ 혹은 ‘Special Competency 행 추가’ Button을 Click하면 행이 한 줄 추가되며, 2항~5항의 순서를 통해 역량을 추가할 수 있다.
    7. 화면 하단의 저장 Button을 Click하면 Competency가 생성되었다는 Message가 나타나며, 확인 Button을
    Click하면 Job Unit별 KSEA 생성화면으로 자동 이동한다.
