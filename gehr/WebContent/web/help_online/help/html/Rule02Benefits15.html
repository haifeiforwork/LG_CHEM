국내신혼여행 경비지원

대 상 : 본인결혼 경조휴가기간 중 국내로 신혼여행을 가는 임직원

지원한도 : 50만원

신청 Process

  • 본인결혼 경조 휴가기간 중 법인개별카드로 신혼여행경비 결제
    * ex) 항공, 렌트, 호텔숙박, 식사
  • ‘국내신혼여행 경비지원 신청서’ 작성 (작성양식은 HR부서에서 수령)
  • 본인 작성 후 > HR부서(본사: HR서비스팀, 지방: 인사지원팀/총무팀/지원팀) 결재
    * 결재 후 원본은 HR부서 보관, 사본은 본인에게 전달
  • 본인이 SAP(ERP) 전표작성 (증빙: 신청서 사본 및 카드매출전표, 청첩장사본)
  • 재경부서 전표 결재 후 본인 등록통장으로 입금

유의사항

  • 법인개별카드를 발급받지 않은 임직원은 본인명의 개인카드를 이용 할 수 있음.
  • 여행경비는 본인의 경조휴가일 내에 사용됨이 원칙임.
  • 여행경비로 보기 힘든 사용처 및 사용내용은 비용처리가 불가 함.
  • 경비지원신청은 결혼일로부터 3개월 이내에 한 함.
    (단, 법인카드 사용 후 전표 처리 시 ‘회계처리 지침’ 및 ‘전표처리 지침’에 따름)
  • 간소화된 작은 결혼식을 지원하는 취지 하에, 국내 신혼여행 전·후 해외로 여행을
    가는 경우 지원 불가함.

필요증빙 : 결혼청첩장 사본, 카드매출전표

문의처 : 본사 - HR서비스팀 (☎ 02-3773-7066)
