장기근속 기념품

지원 대상 : 해당 근속 임직원 (비상근 촉탁 및 일용직 제외)

장기근속 기념품

근속 기념품 기준 비고
35년 120만원 상당의 기념품 근속기념일 휴가 1일
30년 100만원 상당의 기념품 근속기념일 휴가 1일
25년 90만원 상당의 기념품 근속기념일 휴가 1일
20년 70만원 상당의 기념품 재직 중 1회 부부동반 단체여행 또는 여행경비 350만원 중 본인 선택
15년 60만원 상당의 기념품 근속기념일 휴가 1일
10년 40만원 상당의 기념품 근속기념일 휴가 1일, 여행경비 50만원
  • 근속 기념품 지급 시점 : 근속 기념일이 도래한 다음 년도 1월 2일자
  • 근속기념일 휴가 : 해당 근속일로부터 1개월 이내 사용 가능, 전일 공가로 처리
  • 여행경비 : 근속 기념일이 도래한 다음 달 급여에 합산하여 지급
  • 해당 근속이 되는 사원이 시상일 이전에 퇴직할 경우 퇴직 시 근속 포상 (상품권 지급)실시
