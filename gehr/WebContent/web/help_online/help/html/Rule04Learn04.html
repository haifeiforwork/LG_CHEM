외국어 성적 등록방법

성적 등록

신청 방법

  • 신청 메뉴 : G-Portal > HR > Global HR Portal > 교육 > Global Academy > 외국어 성적 등록
  • 담 당 자  : 직무역량개발팀(교육지원P) 신은비 사원
    ※ 유효기간 內 성적만 등록 신청 가능함.(시험응시일로부터 2년)
  • 검정일 입력시 : 실제 검정일을 입력바람
  • 성적 스캔 본 첨부시 : 별도 서류 제출 필요없음 (HSK,TOEFL 은 반드시 담당자에세 원본 성적표 송부요망)
  • LGA-LAP 은 일괄 업로드 됨(별도 등록 필요 없음)
