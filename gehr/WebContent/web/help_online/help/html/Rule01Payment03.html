휴가

연차 휴가 = 기본 연차 15일 + 근속 가산 휴가

기본 연차

  • 근속 1년 이상 근무자로서 년간 소정근로일수 80% 이상 근무시 15일 부여

근속 가산 휴가

  • 3년 이상 계속 근로 시 최초 1년 초과하는 근속년수 매2년마다 1일의 가산휴가 부여 ( 가산휴가를 포함한 총 휴가일수는 재직 중 25일을 초과할 수 없음. )

연차 휴가 생성일 : 매년 12월 21일 부

휴가 예외자

근속 1년 미만자

  • 입사 년도 : 1개월 개근 시 1일의 사전부여 휴가 부여 (근속 1년 시점까지)
  • 익년도 연차휴가 부여 기준 : 근무일수에 비례하여 휴가 생성함

휴/복직자

  • 익년도 연차휴가 부여 기준 : 근무일수에 비례하여 휴가 생성함
