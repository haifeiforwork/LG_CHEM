◈ 세액공제 : 1. 근로소득세액공제 + 2. 주택자금이자세액공제

 

 1. 근로소득세액공제

     산출세액 50만원 이하분 * 55% + 산출세액 50만원 초과분 * 30% 공제 (한도 50만원)

 2. 주택자금이자세액공제

     ① 공제요건 : 아래의 요건을 모두 갖추어야 함

       - 무주택세대주 또는 1주택만을 소유한 세대주
         (무주택세대주 등의 해당여부는 미분양주택 취득시기를 기준으로 판단)

       - 1995.11.1 ~ 1997.12.31까지 미분양주택을 취득(매매계약 체결후 계약금을 납부한 경우 포함)
         단, 서울지역의 미분양주택을 취득시에는 제외됨

       - 당해 주택의 취득과 관련해 1995.11.1이후 차입한 차입금에 대한 이자상환액

     ② 공제금액 : 당해년도 이자상환액의 30%

     ③ 농어촌특별세 징수 : 주택자금이자세액 공제액의 20%를 징수함(전산자동반영)

     ※ 단, 장기주택저당차입금/ 주택취득 차입금상환과 중복될 경우 선택하여 적용

      ☞ 제출서류 : 미분양주택확인서(주택소재지관할 시장, 군수, 구청장이 발행한 것),
                         차입금이자상환액증명서(금융기관발행), 매매계약서 및 등기부등본
